APP就是一个项目的一些功能模块

1.3.1创建一个APP

python manage.py startapp cmdb  # 创建一个名cmdb的APP
* cmdb              # app目录
   * migrations     # 数据修改表结构的目录
   * admin.py       # Django为我们提供的后台管理
   * apps.py        # 配置当前app
   * models.py      # ORM，写指定的类通过命令创建数据库结构
   * tests.py       # 单元测试
   * views          # 业务代码
